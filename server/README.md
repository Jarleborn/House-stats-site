## HouseStats

### Instalation
  * Clona ner repot
  * Öppna en terminal och skriv

        npm install

### Starta appliaktionen
  Linux & Mac:
  * öppna en terminal i root mappen, och skriv

        ./tools/start.sh
Windows
  * Öppna mappen server
  * Öppna en terminal och skriv

        npm run Starta
  * Öppna sedan mappen Client

  * Öppna en ny terminal och skriv

        npm run Starta
